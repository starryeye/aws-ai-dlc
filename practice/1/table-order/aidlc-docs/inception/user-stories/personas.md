# 테이블오더 서비스 - 사용자 페르소나

---

## Persona 1: 고객 (Customer)

| 항목 | 설명 |
|---|---|
| 이름 | 김민수 (대표 고객) |
| 역할 | 매장 방문 고객 |
| 나이대 | 20~50대 |
| 기술 수준 | 초~중급 (스마트폰/태블릿 기본 사용 가능) |

**목표**:
- 대기 시간 없이 빠르게 주문하고 싶다
- 메뉴를 쉽게 탐색하고 원하는 것을 찾고 싶다
- 주문 상태를 확인하고 싶다

**동기**:
- 편리하고 직관적인 주문 경험
- 직원 호출 없이 자율적으로 주문

**불편 사항 (Pain Points)**:
- 복잡한 UI나 작은 버튼
- 주문 후 상태를 알 수 없는 상황
- 이전 고객의 주문 내역이 보이는 경우

**사용 환경**:
- 매장 테이블에 설치된 태블릿
- 터치 기반 조작
- 브라우저 웹 UI

---

## Persona 2: 매장 관리자 (Admin)

| 항목 | 설명 |
|---|---|
| 이름 | 박서연 (대표 관리자) |
| 역할 | 매장 운영자 / 점주 |
| 나이대 | 30~60대 |
| 기술 수준 | 중급 (PC/태블릿 업무 사용 가능) |

**목표**:
- 들어오는 주문을 실시간으로 파악하고 싶다
- 테이블별 주문 현황을 한눈에 보고 싶다
- 메뉴를 쉽게 관리하고 싶다

**동기**:
- 운영 효율성 향상 및 인건비 절감
- 주문 누락 방지
- 매장 상황 실시간 파악

**불편 사항 (Pain Points)**:
- 주문이 들어왔는데 놓치는 상황
- 테이블별 정산이 복잡한 경우
- 메뉴 변경이 번거로운 경우

**사용 환경**:
- 카운터 PC 또는 태블릿
- 브라우저 기반 관리자 화면
- 장시간 연속 사용 (영업 시간 동안)
